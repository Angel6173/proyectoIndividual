# TaskFlow - Gestor de Tareas Personal

**URL del proyecto hospedado:** (https://proyectoindividual-noe5.onrender.com)

**Descripción breve:**  
TaskFlow es una aplicación web para organizar tareas de forma simple y visual. Permite registrar usuarios, crear, editar, completar y eliminar tareas con prioridad, categoría y fecha límite. 
Incluye un calendario interactivo para visualizar tareas por fecha y un panel de administración exclusivo para gestionar usuarios y estadísticas.

**Tecnologías utilizadas:**
- Python (Flask)
- HTML
- CSS
- JavaScript (Vue.js y FullCalendar)
- SQLite 
- Render.com para hosting
